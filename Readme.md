# Cybersecurity Incidents & Data Breach Dashboard  
An interactive Streamlit dashboard to visualise global cybersecurity incidents, data breaches, key attack patterns, industry impact, and financial correlations.

---

## üéØ Introduction  
Cyber-threats are increasing rapidly globally, and organisations must understand vulnerabilities, attack patterns, and industry-specific risks.  
This dashboard brings together real-world cybersecurity incident data (2024) and provides insights through interactive visualisations.

The dashboard helps analyse:
- Global incidents and breaches  
- Industry-wise impact  
- Common attack vectors  
- Recovery time & financial correlations  
- Trends over time  

Built with **Streamlit**, it is simple, fast, and perfect for both learning and professional data analysis use.

---

## ‚úÖ Features  
- **Interactive world map** to view breaches by country/region  
- **Industry-specific analysis** (finance, healthcare, retail, technology, etc.)  
- **Attack vector insights** (phishing, ransomware, malware, insider threat, etc.)  
- **Financial correlation analysis** between breach severity and cost  
- **Time-series trend visualisation**  
- **Downloadable charts & data exports**  
- User-friendly interface powered by **Streamlit**
